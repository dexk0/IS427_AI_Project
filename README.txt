Developed in VSCode and run on python 3.10.6

Load the code into a code editor and run.
There will be a prompt if you want to run the initial run of the Gradient Boosting 
- This is an instance of Gradient Boosting without Hypertuning, and what I used to get a baseline
input "y" or "Y" to run it, or anything else to skip to the part with hypertuning

To modify the parameters of the hypertuning, change the values in the "LR" matrices
